- אני סוב (ani sob) - I return

- אנחנו סובים (anachnu sovim) - we return

- הם סובים (hem sovim) - they return (masculine)
- הן סובות (hen sovot) - they return (feminine)

Description: סוב means "to return" or "to come back" in Hebrew. It is used to describe the action of going back to a previous location or state. The word can be used in various contexts, such as returning home, returning to a previous state of mind, or returning an object to its original place.
